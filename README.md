# BankDash: Modern Banking Interface

A high-fidelity, responsive banking dashboard built from a Figma design. This project focuses on clean UI/UX, financial data visualization, and seamless navigation using **HTML5** and **Bootstrap 5**.

The goal of this project was to convert a complex Figma wireframe into a pixel-perfect, browser-ready interface. It provides a centralized view for users to manage their finances, track expenses, and monitor card activities.

* **Transaction History:** A clean list of recent activities for financial tracking.
